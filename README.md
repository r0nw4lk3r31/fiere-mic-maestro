# Fiere Mic Maestro

An open mic management app built with React, TypeScript, and local storage.

## Features

- Artist signup for open mic performances
- Customer view of the lineup
- Admin dashboard for managing the lineup
- Simple authentication with hardcoded credentials

## Admin Login

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- localStorage for data persistence

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Data Storage

This app uses browser localStorage for data persistence. All artist signups and admin changes are stored locally in your browser.
